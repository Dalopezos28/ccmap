🔧 OPCIÓN 1: Instalación Automática (Recomendado)
──────────────────────────────────────────────────

Windows:
  1. Doble clic en: setup.bat
  2. Seguir las instrucciones
  3. ¡Listo!

Linux/macOS:
  1. Abrir terminal en el directorio del proyecto
  2. Ejecutar: bash setup.sh
  3. ¡Listo!


📝 OPCIÓN 2: Instalación Manual
──────────────────────────────────────────────────

PASO 1: Crear y activar entorno virtual
  Windows:
    python -m venv venv
    venv\Scripts\activate

  Linux/macOS:
    python3 -m venv venv
    source venv/bin/activate

PASO 2: Instalar dependencias
  pip install -r requirements.txt

PASO 3: Aplicar migraciones
  python manage.py migrate

PASO 4: Crear superusuario
  python manage.py createsuperuser
  
  Usuario sugerido: admin
  Email: admin@comedores.com
  Contraseña: (elige una segura)

PASO 5: Poblar datos de ejemplo
  python manage.py poblar_comedores
  
  Cuando pregunte, escribe: s

PASO 6: Iniciar servidor
  python manage.py runserver

🐛 PROBLEMAS COMUNES
──────────────────────────────────────────────────

❌ Error: "No module named 'django'"
  Solución: Activa el entorno virtual
    Windows: venv\Scripts\activate
    Linux/macOS: source venv/bin/activate

❌ Error: "GDAL is not installed"
  Solución: Comentar djangorestframework-gis en requirements.txt
  
❌ Error: "PostGIS extension not found"
  Solución: En PostgreSQL ejecutar:
    CREATE EXTENSION postgis;

❌ Error de conexión a base de datos
  Solución: Verificar credenciales en archivo .env
